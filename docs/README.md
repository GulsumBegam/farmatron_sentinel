# 🌾 Farmatron Sentinel

### Smart Fence with Animal Detection using IoT & Machine Learning

> An intelligent system that protects crops by detecting animal intrusion in real-time.

---

## 🚀 Project Overview

Farmatron Sentinel is a smart agricultural protection system that detects animal intrusion using an **ESP32-CAM** and a trained **Machine Learning model**.

When an animal is detected:

* 🔔 Buzzer is activated to scare the animal
* 📱 Alert is sent to the farmer via Blynk app

---

## 🎯 Key Features

* 📷 Real-time animal detection
* 🧠 ML model using Edge Impulse
* 📡 IoT alerts via Blynk
* 🔔 Instant buzzer response
* ⚡ Low-cost and efficient
